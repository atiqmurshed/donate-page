What is the document object Model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents a document's structure in memory. It allows programs to interact with the document's content, style, and structure

How do you select an element from the DOM?

Using: getElementById, getElementByClassName, querySelectorAll, etc.

What is event delegation in the context of the dom, and why is it useful?

Event delegation in the DOM is a JavaScript technique that uses event propagation to handle events on child elements by attaching a single event listener to a parent element. useful: When an event occurs on a child element, it bubbles up the DOM tree The parent element's event listener handles the event The event listener identifies the source of the interaction by examining the event object's target property
